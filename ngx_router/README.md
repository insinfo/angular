The [AngularDart][docs] router enables navigation from one view to the next as
users perform application tasks.

Part of the `ngx_*` family maintained by the [insinfo/angular][repo] fork
(continuation of the original `angular_router`/`ngrouter` package).

### Resources

* Documentation: [routing guide][router_guide], [API reference][dartdoc]
* Source code and issues: [github.com/insinfo/angular][repo]
* Core framework: [ngx_dart](https://pub.dev/packages/ngx_dart)
* Migrating from `ngrouter`? See the [migration guide][migration].
